# MatrixCalculator
  
Калькулятор позволяет выполнять основные операции с матрицами:  
1. Искать определитель
2. Искать обратную матрицу
3. Транспонировать матрицу
4. Искать ранг матрицы
5. Умножать матрицу на скаляр
6. Возводить матрицу в степень
7. Умножать матрицы
8. Сложение матриц
9. Вычитание матриц
